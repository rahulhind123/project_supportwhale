
For build and run the application 

For MySQL Database
===================

Read_mysql.txt file and run all query given in sql/mysql_db_data.sql file on database then run below command
java -jar mysql_db_assignment.jar

For H2 Database (In Memory Database)
================

Run jar file under jars folder 
java -jar h2_db_assignment.jar

Read readme_h2.txt and run all query given in sql/h2_db_data.sql file
